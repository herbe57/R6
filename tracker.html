Peur de télécharger un virus? Suis ses étapes pour rester en sécutité !

Commence par créer un fichier .txt

Ensuite copie le texte suivant et colle le dans ce fichier texte:

@echo off && mode con: cols=35 lines=3
set /a n=%random%%%9
set /a n=n+1
color 0%n%
title R6tracker
echo.
set /p player= Pseudo :
start chrome --new-window --incognito "https://r6.tracker.network/profile/psn/%player%"

Finalement, renomme le fichier de sorte qu'il ne soit plus un fichier .txt mais .bat
